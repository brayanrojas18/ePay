# ePay

* rest_wallet: 

    Api REST que funciona como puente entre el cliente y la bd(soap)
    Levantar: 
        1. npm install
        2. node index.js

* soap_wallet: 
    
    SOAP que establece conexion entre la bd y el api para enviar datos al cliente.
    Levantar:
        1. composer install
        2. php artisan serve
